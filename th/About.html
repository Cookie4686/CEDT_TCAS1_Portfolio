ที่มาของเว็บไซต์

ในตอนม.5 ผมเคยได้ยินความคิดที่ว่าถ้าจะเข้าคณะเกี่ยวกับคอม ลองทำเป็นเว็บไซต์ส่งเขาไปสิ ผมก็ยังไม่แน่ใจว่าจะทำดีไหม แต่พอผมอยู่ในชั้นม.6 ผมก็ได้ตัดสินใจแล้วว่าจะเข้าเรียนวิศวะฯหลักสูตร CEDT แต่ผมก็ไม่มีผลงานอะไรโดดเด่นเลย แต่ว่าผมก็ต้องการที่จะเข้าในรอบนี้ ดังนั้นผมจึงเริ่มต้นทำเว็บไซต์นี้มาตั้งแต่ปลายเดือน มิ.ย. ปี 2566 เพื่อจะได้มีผลงานขึ้นมาพิสูจน์ตัวเองบ้าง และหวังว่าเว็บไซต์นี้ช่วยให้ผมที่จะผ่านเข้ารอบนี้ไปได้ เพื่อที่ผมจะได้ใช้เวลาที่ได้มาเพิ่มจากการที่ไม่ต้องเตรียมสอบไปเรียนรู้ในด้านต่างๆ และทำ side-project อื่นๆที่ผมสนใจในอนาคตได้ต่อไป

ขั้นตอนการทำ

เว็บไซต์นี้เป็น Static Web Page ถูกสร้างขึ้นด้วยภาษา HTML , CSS และ JS ซึ่งไม่ได้ใช้ Framework และ AI ในการทำ เนื่องจากผมต้องการฝึกฝนจากพื้นฐานก่อน โดยคติในการทำหลักของผมคือ “Simple yet Elegant” ซึ่งผมจะเน้นให้เรียบง่ายแต่สวยงามตามสไตล์คล้ายๆ Minimalism เพราะผมไม่ได้มีทักษะการทำเว็บไซต์ให้สวยงามอลังการมากนัก โดยเว็บไซต์นี้ผมได้ใช้เวลาในการทำอยู่ประมาณ 4 เดือน ซึ่งมีรายละเอียดดังนี้

ปลายเดือนมิ.ย. - เดือนก.ค.
ในช่วงแรกของการทำเว็บไซต์ ผมยังไม่ได้มีเนื้อหาหรือแผนผังเว็บไซต์ว่าจะทำอะไรบ้าง ผมจึงเริ่มทำแถบเมนูด้านบนก่อน (Navigation Bar) ซึ่งผมทำเป็นแบบ Responsive แบบง่ายๆ สำหรับในคอมเป็นแถบด้านบน และในมือถือเป็นแถบด้านข้าง ที่มีปุ่มเมนูไว้กดเปิด ในส่วนหัวของเว็บไซต์ (Header) ก็จะมีข้อความต้อนรับ หลักสูตรและโครงการที่จะเข้า เหมือนในแฟ้มสะสมผลงานแบบ A4

เดือน ส.ค.
ในช่วงนี้ผมก็พยายามทำให้โค้ดของผมอ่านง่าย ใช้บรรทัดน้อยลง เพิ่มส่วนของการแนะนำตนเอง ประวัติการศึกษา ทักษะความสามารถ และใส่ Animation ให้แต่ละส่วนเมื่อผู้ใช้เลื่อนจอลงมาเรื่อยๆ (แต่ในส่วนอนิเมชั่นก็เอาออกตอนหลัง)

เดือน ก.ย.
ผมคิดว่า NavBar ที่มีอยู่ยังดูธรรมดาเกินไป ผมเลยทำการเพิ่มโหมดสว่าง/โหมดมืดเข้ามา เพิ่มโลโก้เข้ามาในแถบด้านซ้ายบน และเปลี่ยนให้แถบ Navigation ในมือถือเป็นแบบเต็มจอแทน

ต้นเดือนต.ค.
ในเดือนนี้เป็นช่วงปิดเทอมและผมก็ทำแฟ้มสะสมผลงานแบบ A4 ได้คร่าวๆแล้ว ผมจึงคิดแผนผังเว็บไซต์ใหม่ให้เหมือนใน A4 และเริ่มมีการวางแผนเป็นระบบระเบียบมากขึ้น โดยมีการจดโน้ตไว้ว่าจะทำอะไรยังไงบ้าง และเริ่มวางไฟล์ต่างๆให้พร้อมกับการเผยแพร่เว็บไซต์จริงๆ

กลางเดือน ต.ค.
ในวันที่ 18 ต.ค. มีการประกาศเกณฑ์แฟ้มสะสมผลงานออกมา ผมก็ปรับแก้ในส่วนของ A4 ให้ตรงตามเกณฑ์ และเตรียมเนื้อหาที่จะใส่เข้าไปในเว็บไซต์ ช่วงนี้ผมก็ใกล้เปิดเทอมแล้ว ผมจึงวางแผนว่าจะทำส่วนที่จำเป็นก่อน ส่วนเสริมหากทำไม่ทันก็ให้ตัดทิ้งไป

ปลายเดือน ต.ค. - พ.ย.
ในช่วงสุดท้ายที่ Deadline กำลังใกล้เข้ามาผมได้ทดสอบการเผยแพร่เว็บไซต์ พยายามเรียบเรียงเนื้อหาต่างๆที่ใช้ทั้งในเว็บไซต์และA4 และแปลเนื้อหาที่เรียบเรียงแล้วเป็นภาษาอังกฤษด้วย
